This is a simple CRUD application built with ASP.NET Minimal APIs for the backend and HTML, Bootstrap, and JavaScript for the frontend. It demonstrates how to create, read, update, and delete data using a RESTful API, and dynamically update the user interface. This project is designed for beginners to learn modern web development concepts.
